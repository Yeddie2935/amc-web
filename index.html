Create 50 launch problems using original wording. This avoids copyright risk and lets the platform prove the learning flow.

Phase 2: Link-first AMC archive

Store metadata, category, difficulty, and official/source links. Keep the copyrighted statement off-site unless permission is obtained.

Phase 3: Licensed/imported content

When you have permission or a public-domain-compatible source, import full problem text using the JSON schema in data/problem-schema.md.
